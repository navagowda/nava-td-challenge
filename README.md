# NAVA movable drawing toolbar update

Replace `src/app/chart/page.tsx` in your project with the included file.

## Included
- Drag the drawing toolbar anywhere inside the chart using the grip handle.
- Hide the toolbar with the eye-off button.
- Restore it with the small Drawings button.
- Toolbar position and visibility are saved in the browser and Supabase workspace.
- Hiding the toolbar automatically returns the chart to normal interaction mode.
